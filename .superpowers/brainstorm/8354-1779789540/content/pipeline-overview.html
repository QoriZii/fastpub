Key Question

What kind of output video are you imagining?

A

Screencast Style

Looks like someone presenting slides — cursor moves to highlight points, elements appear progressively, smooth transitions between slides. Like a polished screen recording of a presentation.

B

Motion Graphics Style

Cinematic text animations, kinetic typography, figures fly in with spring physics. More like a YouTube explainer video (Kurzgesagt-lite). No cursor — everything animates on its own.

C

Hybrid

Slide-based canvas with a cursor for pointing/highlighting, but elements also animate in (fade, slide). A middle ground — feels like an animated presentation with a virtual presenter's cursor.

Animation Primitives

Shared building blocks used across scene types

↑
FadeIn
Opacity 0 → 1 with optional Y offset. For headlines, body text.
↪
SlideIn
Translate from edge with spring easing. For images, cards.
●
ScaleReveal
Scale 0.8 → 1 with opacity. For key metrics, emphasis.
✎
TypeWriter
Character-by-character text reveal. For hook headlines.
⇄
CrossFade
Smooth transition between scenes. Overlap last/first frames.
★
Stagger
Sequential reveal of list items with delay. For key points.

Scene Type → Animation Mapping

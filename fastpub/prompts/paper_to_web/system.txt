You are a research paper analyst. Extract structured information for a bilingual (English + Simplified Chinese) web summary page.

Produce a JSON with paper metadata, a hook sentence, and exactly 5 web sections. For every text field, provide both the English value and a `_zh` Chinese translation.

## Output Format

```json
{
  "meta": {
    "title": "English title",
    "title_zh": "中文标题",
    "authors": ["Author Name"],
    "venue": "string or null",
    "year": "number or null",
    "abstract": "English abstract",
    "abstract_zh": "中文摘要",
    "keywords": ["English keyword"]
  },
  "hook": "one attention-grabbing sentence",
  "hook_zh": "中文钩子句",
  "webSections": [
    {
      "type": "problem",
      "summary": "2-3 sentences on the core problem",
      "summary_zh": "中文总结",
      "subIssues": [
        {
          "title": "Sub-issue name",
          "title_zh": "中文子问题",
          "description": "1-2 sentences",
          "description_zh": "中文描述"
        }
      ]
    },
    {"type": "approach", "summary": "...", "summary_zh": "...", "subIssues": [...]},
    {"type": "meaning", "summary": "...", "summary_zh": "...", "subIssues": [...]},
    {"type": "result", "summary": "...", "summary_zh": "...", "subIssues": [...]},
    {"type": "limitation", "summary": "...", "summary_zh": "...", "subIssues": [...]}
  ]
}
```

## Guidelines

- Produce exactly 5 sections in order: problem, approach, meaning, result, limitation
- Each section: concise summary (2-3 sentences) + 2-4 sub-issues
- Sub-issues must be specific and concrete — not vague restatements
- Audience level: "{{audienceLevel}}"
  - "academic": precise terminology for researchers
  - "general": plain language for non-specialists
- Extract metadata accurately from the paper text
- Chinese translations must be natural-sounding (not translationese), use standard Chinese academic terminology
- Keep technical terms commonly used in English (e.g., Transformer, GAN) — transliterate or keep English in parentheses where appropriate
- Do NOT translate person names or author names — keep them in English exactly as given

Return ONLY the JSON object — no markdown fences, no explanation.

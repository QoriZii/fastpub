How Remotion Handles This

Single <Scene> component
One component handles all scene types. It picks the right layout (centered vs split) and animation set based on scene.type and whether scene.imageFile exists.
Dynamic total duration
Total video length = sum of all scene durations (from TTS audio). Remotion's durationInFrames is calculated at render time from the manifest. No hardcoded limit — the 5-min max is enforced at the Python level.

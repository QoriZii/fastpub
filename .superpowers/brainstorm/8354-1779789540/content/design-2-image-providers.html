Image Prompt Strategy

LLM generates prompts
The scene script LLM call produces an imagePrompt per scene — a detailed visual description optimized for image generation. This is part of the scene script output, not a separate LLM call.
Consistent visual style
A style prefix is prepended to every prompt (e.g., "Clean, modern infographic style, flat design, muted professional colors.") to ensure visual consistency across scenes.

CLI Usage

# Slides only (scene script + image gen + HTML)
$ fastpub render analysis.json -f slides
# Video only (runs slides stage internally, then TTS + Remotion)
$ fastpub render analysis.json -f video
# Both (slides stage runs once, shared assets)
$ fastpub render analysis.json -f slides,video
# Full pipeline from PDF
$ fastpub go paper.pdf -f slides,video --image-provider xai

Output Structure

output/
  paper-name.analysis.json       # from analyze step
  paper-name.slides.html         # HTML slide deck
  paper-name.scenes.json         # scene script (shared by slides + video)
  paper-name.mp4                 # final video
  paper-name/
    images/                      # AI-generated images
      scene-1.png
      scene-2.png
      ...
    audio/                       # TTS narration (video only)
      scene-1.mp3
      scene-2.mp3
      ...
    manifest.json                # Remotion input (video only)

Internal Dependency Flow

render_slides(doc)
→ generate_scene_script(doc) → scenes.json
→ generate_images(scenes) → images/
→ build HTML slide deck
render_video(doc)
→ render_slides(doc) # if not already done
→ generate_tts(scenes) → audio/
→ build_manifest(scenes, audio, images) → manifest.json
→ npx remotion render → output.mp4
